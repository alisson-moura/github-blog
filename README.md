## 🚀 Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/alisson-moura/github-blog
```

Entre no diretório do projeto

```bash
  cd github-blog
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm run dev
```
